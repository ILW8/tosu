

## Requirements

- `Typescript` >=5.3.3
- `Node.js` >=20.11.1
- `Rust` >= any
- `pnpm` >= 8

<br />

1. Clone repository

```
git clone https://github.com/KotRikD/tosu.git
```

2. Go to project folder
```
cd tosu
```

3. Install pnpm
```
npm install -g pnpm
```

4. Install dependencies
```
pnpm install
```

5. To run tosu in dev mode
```
pnpm run start
```


6. Compile tosu
```
pnpm install && pnpm build
```

7. Go to `/tosu/packages/tosu/dist`, and there is your's tosu build
